Online Retail Customer Segmentation Analysis¶

Project Goal¶

The goal of this project is to analyze customer purchase behavior using transaction data and identify high-value, regular, and at-risk customers. The results can support targeted marketing, customer retention, and revenue growth strategies.

2. Customer Segmentation (RFM)¶

RFM analysis is used to summarize customer behavior at the individual level:

  • Recency: how recently a customer made a purchase
  • Frequency: how often a customer purchased
  • Monetary: how much the customer spent

These metrics help group customers into meaningful business segments.

# Recency: smaller is better → reverse score
rfm["R_score"] = pd.qcut(
    rfm["Recency"],
    q=4,
    labels=[4, 3, 2, 1],
    duplicates="drop"
)

# Frequency: larger is better → use rank to stabilize
rfm["F_score"] = pd.qcut(
    rfm["Frequency"].rank(method="first"),
    q=4,
    labels=[1, 2, 3, 4],
    duplicates="drop"
)

# Monetary: larger is better
rfm["M_score"] = pd.qcut(
    rfm["Monetary"],
    q=4,
    labels=[1, 2, 3, 4],
    duplicates="drop"
)

def segment_customer(row):
    if row["RFM_Label"] == "444":
        return "VIP"
    elif row["R_score"] >= 3 and row["F_score"] >= 3:
        return "Loyal"
    elif row["R_score"] <= 2 and row["F_score"] <= 2:
        return "At Risk"
    else:
        return "Regular"

rfm["Segment"] = rfm.apply(segment_customer, axis=1)
